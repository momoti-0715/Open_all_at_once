exeファイルを実行すると下で設定したものがまとめて開きます
(windowsでの動作を想定)

Shortcutフォルダ内のexeファイルと、Settingフォルダ内のjsonは名前で連動しています
実行したい設定のファイル名と同じ名前のshortcutフォルダ内のexeファイルを実行すると、実行されます

パスは全て実行ファイルの絶対パスで記述(バックスラッシュはエスケープ処理してください)

※コマンド実行は自己責任で


以下はjsonの記載ルール

{
    urlを開く
    "web_open": {
        "browser": ブラウザを指定する場合はパスを記載(nullの場合はデフォルトのブラウザ),
        "url_list": [
            url1,
            url2,
            ...(指定しない場合はnull)
        ]
    },
    コマンドプロンプトを開く
    "cmd_open": {
	"p_c_list":[ "開きたい数だけ{}で追加する"
            {
                "path": カレントディレクトリのパス(指定しない場合はnull),
                "command": 事前にコマンドを使用する場合は記載(使用しない場合はnull) パイプは使用不可
            },
            {
                ...(さらに開きたい場合は↑に従って記述する)
            }
		開かない場合は{}のみで
        ]
        
    },
    タスクを開く
    "task_open": {
        "task_list": [
            タスクのパス1,
            タスクのパス2,
            ...(指定しない場合はnull)
        ]
    }
}
